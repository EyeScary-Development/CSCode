CSCode from EyeScary Development, simplifying user configuration for Ace code editor

        
Note: we are migrating to keyboard shortcuts and soon some buttons will be more hidded, here are the keyboard shortcuts for some of these:

Ctrl + s : download the file
Ctrl + i : open a local file
Ctrl + Shift + Space : find mode (find all instances of a string you type in)
 Ctrl + . : find and replace (replace all instances of an inputted string with another one) 
 Alt + , : open settings (don't worry, though, the settings button will not be replaced, this is an alternative way of getting to your settings)
Alt + s : save online
Alt + i : load online save
